Privacy Policy

Last updated: January 2025

Information We Collect

The NANO MCP Server processes cryptocurrency transaction data as part of its core functionality. This includes:

  • NANO wallet addresses
  • Transaction amounts and blockchain data
  • API usage logs for debugging and performance monitoring

How We Use Your Information

We use the collected information to:

  • Process NANO cryptocurrency transactions
  • Provide API services and documentation
  • Monitor system performance and debug issues
  • Ensure security and prevent unauthorized access

Data Storage and Security

Your data is processed temporarily to fulfill API requests. We implement appropriate security measures to protect your information:

  • Private keys are never stored permanently
  • All communications use secure protocols
  • Access to systems is restricted and monitored

Third-Party Services

This service may interact with external NANO RPC nodes and blockchain networks to process transactions. Please review the privacy policies of these third-party services.

Your Rights

You have the right to:

  • Know what personal information is being processed
  • Request deletion of your data where applicable
  • Object to processing of your personal information

Changes to This Privacy Policy

We may update this privacy policy from time to time. We will notify you of any changes by posting the new privacy policy on this page.

Contact Us

If you have any questions about this Privacy Policy, please contact us through our GitHub repository or API documentation.
